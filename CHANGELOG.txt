CDeTDS — Changelog
==================

v1.0.2 (June 2026) - Compliance fixes, app rename, UI improvements
--------------------------------------------------------------------
COMPLIANCE (CRITICAL)
  * Fix Section 194H threshold: Rs.15,000 → Rs.20,000 per Finance Act 2025
  * Fix Section 194I threshold: Rs.6,00,000 annual → Rs.50,000 per month per Finance Act 2025
  * Add Section 194P (senior citizen 75+ bank TDS, cess=4) to built-in rules
  * Add Section 206AA (higher TDS when PAN unavailable) to built-in rules
  * Fix Professional Tax (Section 16(iii)) now deducted in new regime taxable income
    (CBDT Circular 04/2022 — deductible in both old and new regimes)
  * TDS rules version bumped to 2026-27-20260601 — auto-corrects all installed DBs on startup

APP RENAME
  * Renamed from TDS Pro to CDeTDS throughout (display name, namespaces, folders)
  * AppData path migrated: %AppData%\TDSPro → %AppData%\CDeTDS (auto on first launch)
  * Documents path: Documents\TDSPro → Documents\CDeTDS

FVU FIXES
  * Fix T-FV-3070: BH[47] challan total now uses same deductee-assignment as CD[27]
  * Fix T-FV-1041/6138: auto-upgrade stale FvuPath to 9.4 JAR on startup
  * Fix FVU version warning on migrated databases
  * Fix deductee with stale ChallanNo silently dropped from FVU output
  * CSI Download step now correctly shown before Validate with FVU in Step guide

PAYROLL
  * Fix LOP calculation: now uses Gross ÷ 30 (was Basic ÷ 30) — standard Indian payroll practice
  * Fix deduction schedule balance: actual varDed amount used, supports extra repayments per month
  * Fix salary slips not updating after bulk save or single save
  * Fix Salary Slips tab auto-syncs to current month on tab switch

DATABASE
  * Fix DB migrations now run synchronously before window opens (prevented Payroll crash on fresh install)
  * Add last_posted_amt column to deduction_schedules for idempotent re-posting

UI
  * Help drawer: full-text search across all FAQ topics
  * NIL return dialog: show Rs.100/day penalty warning u/s 234E
  * Challan reconciliation: actionable fix instructions shown on mismatch
  * FY Rollover: clarify run timing (once in April after year-end)
  * Login: warn to change default password immediately
  * Deductee Master, Audit Log, Users: fixed scroll clipping (flex layout)
  * TDS Rules Reference: column widths optimised, Active+Actions merged
  * Help topics updated: CDeTDS paths, LOP explanation, salary slips tab sync

INSTALLER
  * Fix bundled JRE path in installer (was not included in v1.0.1 installer)
  * Fix FvuPath and JavaPath registry entries now point to correct JAR/exe
  * Installer renamed: TDSPro_Setup → CDeTDS_Setup

TESTS
  * 281 tests — added LOP, deduction schedule balance, professional tax tests
  * All tests pass
